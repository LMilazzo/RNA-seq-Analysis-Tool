pathfindR: Pathway Enrichment Analysis in R

The pathfindR package enhances differential gene expression (DGE) analysis by identifying biological pathways significantly enriched among differentially expressed genes (DEGs). It pairs seamlessly with DESeq2 for comprehensive insights.

What is Pathway Enrichment Analysis?

Pathway enrichment analysis evaluates groups of related genes to identify pathways significantly impacted by experimental conditions. This approach provides higher-level insights into affected biological processes, translating gene-level changes into meaningful biological interpretations.

How pathfindR Works with DESeq2

Using DESeq2 for differential expression analysis produces a list of DEGs based on significant fold changes. pathfindR then takes these DEGs and performs pathway enrichment analysis to identify which pathways are likely affected by the observed gene expression changes:

  • DESeq2: Detects significant changes in gene expression, generating a DEG list with fold change and p-values.
  • pathfindR: Performs enrichment analysis on DEGs to identify significant pathways, using databases like KEGG or Reactome.

Features of pathfindR

pathfindR provides several features that make pathway enrichment analysis accessible and informative:

  • Comprehensive Pathway Databases: Supports multiple pathway databases for versatile analysis.
  • Visual Representation: Offers bubble plots and other visual outputs to facilitate result interpretation.
  • Data Filtering: Built-in filtering improves analysis accuracy by removing low-confidence findings.
